plotypus.utils module¶

plotypus.utils.verbose_print(message, *, operation, verbosity)[source]¶

Prints message to stdout only if the given operation is in the list of verbose operations. If “all” is in the list, all operations are printed.

plotypus.utils.pmap(func, args, processes=None, callback=<function <lambda> at 0x7f23dcb49158>, **kwargs)[source]¶

Parallel equivalent of map(func, args), with the additional ability of providing keyword arguments to func, and a callback function which is applied to each element in the returned list. Unlike map, the output is a non-lazy list. If processes=1, no thread pool is used.

plotypus.utils.make_sure_path_exists(path)[source]¶

plotypus.utils.autocorrelation(data, lag=1)[source]¶

Computes the autocorrelation of the data with the given lag. Autocorrelation is simply autocovariance(data) / covariance(data-mean, data-mean), where autocovariance is simply covariance((data-mean)[:-lag], (data-mean)[lag:]).
